# PDSA Course Work Project

A full-stack application implementing Problem Solving and Data Structures & Algorithms concepts.

## Architecture

- **Frontend**: React with TypeScript
- **Backend**: ASP.NET Core Web API (.NET 9.0)
- **Database**: SQLite with Entity Framework Core

## Prerequisites

- **Node.js** (v18 or higher)
- **.NET 9.0 SDK**

## Getting Started

### Backend Setup

```bash
cd PDSA.API
dotnet build
dotnet run
```

API available at: `http://localhost:5000`  
Swagger UI: `http://localhost:5000/swagger`

### Frontend Setup

```bash
cd pdsa-frontend
npm install
npm run dev
```

Frontend available at: `http://localhost:3000`
